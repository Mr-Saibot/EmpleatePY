# Changelog

Todos los cambios importantes realizados en el proyecto serán documentados en este archivo.

## [1.0.0] - 2026-07-22

### Agregado

* Creación de la página principal.
* Implementación del listado dinámico de proveedores mediante JSON.
* Consumo de datos utilizando Fetch API.
* Buscador de proveedores por nombre.
* Filtro por categoría.
* Ordenamiento por nombre y calificación.
* Página individual para cada proveedor.
* Sistema de inicio de sesión utilizando LocalStorage.
* Proveedor destacado generado automáticamente desde el archivo JSON.
* Botón de llamada mediante enlaces `tel:`.
* Diseño responsive para dispositivos móviles, tablets y computadoras.
* Estados de carga, error y ausencia de resultados.
* Optimización del diseño y experiencia de usuario.
* Evaluación del proyecto utilizando Lighthouse.
* Documentación del proyecto mediante README.
